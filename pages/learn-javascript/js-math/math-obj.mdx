title: "JavaScript Math Object"
date: "30 Jul, 2024"
---

The JavaScript Math object provides built-in methods for performing mathematical operations on numbers. It has static properties and does not require a constructor. Math works with Number data types but not with BigInt.

## Example 1: Math Object Properties

This example uses `Math` object properties to return their values:

```javascript
console.log("Math.LN10: " + Math.LN10);
console.log("Math.LOG2E: " + Math.LOG2E);
console.log("Math.Log10E: " + Math.LOG10E);
console.log("Math.SQRT2: " + Math.SQRT2);
console.log("Math.SQRT1_2: " + Math.SQRT1_2);
console.log("Math.LN2: " + Math.LN2);
console.log("Math.E: " + Math.E);
console.log("Math.PI: " + Math.PI);
```

### Output:
```
Math.LN10: 2.302585092994046
Math.LOG2E: 1.4426950408889634
Math.Log10E: 0.4342944819032518
Math.SQRT2: 1.4142135623730951
Math.SQRT1_2: 0.7071067811865476
Math.LN2: 0.6931471805599453
Math.E: 2.71828...
```

## Example 2: Math Object Methods

Math object methods are used in this example:

```javascript
console.log("Math.abs(-4.7): " + Math.abs(-4.7)); 
console.log("Math.ceil(4.4): " + Math.ceil(4.4)); 
console.log("Math.floor(4.7): " + Math.floor(4.7)); 
console.log("Math.sin(90 * Math.PI / 180): " +
        Math.sin(90 * Math.PI / 180)); 
console.log("Math.min(0, 150, 30, 20, -8, -200): " +
        Math.min(0, 150, 30, 20, -8, -200)); 
console.log("Math.random(): " + Math.random()); 
```

### Output:
```
Math.abs(-4.7): 4.7
Math.ceil(4.4): 5
Math.floor(4.7): 4
Math.sin(90 * Math.PI / 180): 1
Math.min(0, 150, 30, 20, -8, -200): -200
Math.random(): 0.7416861489868538
```
