## What is Map?

A Map in JavaScript stores key-value pairs, allowing keys to be of any data type. Unlike objects, it maintains the insertion order and provides built-in methods to add, get, delete, and iterate over elements efficiently.

## Syntax
```js
new Map();
new Map([iterable_objects]);
```

**Return type**: It returns a new map object.

**Parameters**: An array or iterable object.

## JavaScript Map Properties

A **JavaScript property** is a member of an object that associates a key with a value.

### Instance Properties

| Property     | Description |
|-------------|-------------|
| `constructor` | It is used to return the constructor function of Map. |
| `size` | Returns the number of keys and value pairs stored in a map. |

## JavaScript Map Methods

JavaScript methods are actions that can be performed on objects.

### Instance Methods

| Method | Description |
|--------|-------------|
| `clear()` | Removes all the elements from a map and makes it empty. |
| `delete()` | Deletes the specified element among all the elements present in the map. |
| `entries()` | Returns an iterator object containing all `[key, value]` pairs of each element in the map. |
| `forEach()` | Executes a given function over each key-value pair in the map. |
| `get()` | Returns a specific element among all the elements present in a map. |
| `has()` | Checks whether an element with a specified key exists in a map or not. |
| `keys()` | Returns an iterator object of keys from a given map object. |
| `set()` | Adds key-value pairs to a Map object. |
| `values()` | Returns a new iterator object that contains the value of each element present in the Map. |
