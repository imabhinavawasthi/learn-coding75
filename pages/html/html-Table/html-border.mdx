# HTML Table Borders

HTML tables can have borders of different styles and shapes.

## How To Add a Border

To add a border, use the CSS `border` property on table, `th`, and `td` elements:

```css
table, th, td {
  border: 1px solid black;
}
```

# Table Border in HTML

A table border in HTML refers to the visual boundary around the table and its cells, distinguishing the table content from the rest of the page. By default, tables don't have visible borders, but you can add them using CSS.

## Example:

```css
table, th, td {
  border: 1px solid black;
}
