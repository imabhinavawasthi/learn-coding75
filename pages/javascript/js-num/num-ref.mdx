## JavaScript Number Reference

JavaScript Numbers are always stored in double-precision 64-bit binary format IEEE 754. The types of number literals you can use include decimal, binary, octal, and hexadecimal.

### Syntax:

```javascript
Number(object);
```

### Example:

```javascript
function func() {   
    // Original string 
    let a = true; 
      
    let value = Number(a); 
    console.log(value); 
} 
func();
```

**Output:**

```
1
```
