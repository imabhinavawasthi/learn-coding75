## JavaScript Number Reference

In JavaScript, numbers are always stored in 64-bit double-precision format (IEEE 754). You can write numbers in different formats like decimal, binary, octal, and hexadecimal.

### Syntax:

```javascript
Number(object);
```

### Example:

```javascript
function func() {   
    // Original string 
    let a = true; 
      
    let value = Number(a); 
    console.log(value); 
} 
func();
```

**Output:**

```
1
```
