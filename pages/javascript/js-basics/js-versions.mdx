JavaScript is a widely used programming language that is lightweight, easy to learn, and works on both browsers (client-side) and servers (server-side). It was created by Brendan Eich in 1995.

In 1997, JavaScript became an official standard called ECMAScript. Since then, new versions have been released regularly, with modern updates named after their release year, such as ECMAScript 2016, ECMAScript 2017, and so on.
