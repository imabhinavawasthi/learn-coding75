JavaScript provides different methods to display output, such as `console.log()`, `alert()`, `document.write()`, and manipulating HTML elements directly. Each method has its specific use cases, whether for debugging, user notifications, or dynamically updating web content. Here we will explore various JavaScript output methods, demonstrating how and when to use them effectively.

There are primarily multiple distinct methods for displaying output in JavaScript. Although there are other methods for output display, they are not advisable, so it is recommended to refrain from using alternative output approaches.

## JavaScript innerHTML Property
The `innerHTML` property is used with HTML elements. JavaScript can select an element using the `document.getElementById(id)` method and then use `innerHTML` to display the output on the selected element.

### Syntax:
```js
document.getElementById("id").innerHTML;
```

### Example:
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <title>JavaScript Output</title>
</head>
<body>
    <h2>Display Output using innerHTML Property</h2>
    <p id="Coding75"></p>
    <script>
        document.getElementById("Coding75").innerHTML = "Hello Geeks!";
    </script>
</body>
</html>
```

## JavaScript document.write() Method
The `document.write()` method is used to write content directly to an HTML document while it is being parsed.

**Note:** If used after the document is fully loaded, it will overwrite the entire page.

### Syntax:
```js
document.write();
```

### Example:
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <title>JavaScript Output</title>
</head>
<body>
    <h2>Display Output using document.write() Method</h2>
    <script>
        document.write("Hello Coders!");
    </script>
</body>
</html>
```

## JavaScript window.alert() Method
The `window.alert()` method displays an alert box with a message and an OK button.

### Syntax:

**Note:** The `window` keyword is optional.

### Example:
```html
<!DOCTYPE html>
<html lang="en">
<head>

</head>
<body>
    <h2>Display Output using window.alert() Method</h2>

    </script>

</html>
```

## JavaScript window.prompt() Method
The `window.prompt()` method displays a dialog box for user input and returns the entered text as a string.

**Note:** This method captures user input but does not directly display output.

### Syntax:
```js
window.prompt();
```

### Example:
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <title>JavaScript Output</title>
</head>
<body>
    <h2>Display prompt box for user input</h2>
    <script>
        let userInput = window.prompt("Please Enter your Input");
        if (userInput !== null) {
            window.alert("Hello, " + userInput + "!");
        } else {
            window.alert("You clicked Cancel or closed the prompt.");
        }
    </script>
</body>
</html>
```
