## JavaScript push() Method

The `push()` method adds one or more elements to the end of an array and returns the new length.

```js
let a = ['Hello', 'Coding75', 'JS'];
a.push('React');
console.log(a);
```

**Output:**

```js
[ 'Hello', 'Coding75', 'JS', 'React' ]
```

## JavaScript map() Method

The `map()` method creates a new array by applying a function to each element.

```js
let a1 = [1, 2, 3, 4, 5];
let a2 = a1.map(elem => elem * 5);
console.log(a2);
```

**Output:**

```js
[ 5, 10, 15, 20, 25 ]
```

## JavaScript reduce() Method

The `reduce()` method applies a function to reduce the array to a single value.

```js
let a1 = [1, 2, 3, 4, 5];
let sum = a1.reduce((acc, elem) => acc + elem, 0);
console.log(sum);
```

**Output:**

```js
15
```

## JavaScript filter() Method

The `filter()` method filters elements based on a condition.

```js
let a1 = [1, 2, 3, 4, 5];
let a2 = a1.filter(elem => elem % 2 === 0);
console.log(a2);
```

**Output:**

```js
[ 2, 4 ]
```
