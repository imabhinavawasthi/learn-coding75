JavaScript `Math` object is used to perform mathematical operations on numbers. `Math` is an inbuilt object that works with `Number` types but does not work with `BigInt`.

## Example

Below example will give you a brief idea of JavaScript math objects:

```javascript
// Return PI value (3.141592653589793) 
console.log(Math.PI);
```

### Output:
```
3.141592653589793
```

This will print the value of Pi on the console.
